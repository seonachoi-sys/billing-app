# CSS / 스타일 규칙

## 스택
- Tailwind CSS 3.4 (유틸리티 클래스)
- Pretendard 폰트
- Recharts (차트)

## 규칙
1. 인라인 스타일 금지 → Tailwind 클래스 사용
2. 커스텀 CSS 필요 시 index.css에 추가
3. 반응형: sm/md/lg 브레이크포인트 항상 고려
4. 색상: Tailwind 기본 팔레트 사용
   - 위험/연체: red-500
   - 경고: yellow-500
   - 완료: green-500
   - 기본: blue-500

## 출력 레이아웃
- 거래명세서 출력 시 print: 클래스 별도 관리
- 화면 UI와 출력 UI 분리
